# SA-MP-CODEAK
ไฟล์สคริปต์ SA-MP สำหรับให้ศึกษาระบบต่าง ๆ

## ระบบทั่วไป
- ลงทะเบียน/เข้าสู่ระบบ (ระบบสมาชิก)
- ฐานข้อมูล MYSQL
- สุ่มกล้องเลื่อนช่วงเข้าสู่ระบบ
- ระบบ Custom OOC (ภาษาไทยจำเป็นต้องมี Plugin CEFix)
